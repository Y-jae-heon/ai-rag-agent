주요 대상:

- API 관련 폴더
- Hook 관련 폴더
- UI 컴포넌트 폴더
- 페이지 및 화면 폴더
- 라우팅 및 네비게이션 폴더
- 상태 관리 폴더
- 공통 라이브러리 폴더
- 리소스 및 다국어 폴더

이 규칙은 **프로젝트 구조와 아키텍처의 일관성 유지**를 목적으로 한다.
