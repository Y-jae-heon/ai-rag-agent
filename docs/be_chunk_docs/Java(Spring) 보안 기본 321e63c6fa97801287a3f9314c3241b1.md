## Title

Java(Spring) 보안 기본

## Rule

### 인증/인가

- 인증은 Spring Security + JWT 를 사용합니다.
- 인가는 AoP를 사용해 제어합니다. 인가는 각 프로젝트의 인가 방법에 따라 구현합니다.

### 암호화

**비밀번호**

- 비밀번호 암호화는 `BCrypt` 단방향 해시를 사용합니다.

**단방향 암호화**

- CI 값과 같이 단방향 암호화만 필요하다면 `HMAC-SHA256` 을 사용합니다.

**양방향 암호화**

- 이름, 주민번호와 같이 양방향 암호화가 필요하다면 `AES-256 + HMAC 인덱스` 를 사용합니다.
